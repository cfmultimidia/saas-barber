# Saas Barber - Sistema de Agendamento

Sistema SaaS para agendamento de serviços em salões de beleza.

## Tecnologias

### Backend
- Node.js + Express
- SQLite
- Socket.io (WebSockets)
- JWT Authentication

### Frontend
- React + Vite
- React Router
- Socket.io Client
- date-fns

## Executar Localmente

```bash
# Backend
cd backend
npm install
npm run seed  # Popular banco com dados de exemplo
npm start

# Frontend
cd frontend
npm install
npm run dev
```

## Deploy

- **Frontend**: Netlify
- **Backend**: Render.com
